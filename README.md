# ドラマー知識検定(仮称)

ドラマー向け雑学クイズアプリのパイロット版。暗記ではなく理解・判断・応用・表現を評価する検定を目指す。

## 現状(パイロット版)

- 13カテゴリ中「1. 基礎知識編」100問のみ実装済み(他12カテゴリはデータ未生成、TOP画面に「近日公開」表示)
- 画像を伴う問題(写真・イラスト・譜面)はアセット未調達のためプレースホルダー表示
- 広告(バナー/インタースティシャル)は実際の広告SDKと未連携のプレースホルダー実装

## 動かし方

`index.html` をブラウザで直接開く(ビルド不要の静的サイト)。

## ファイル構成

- `index.html` — エントリーポイント
- `styles.css` — スタイル
- `app.js` — 画面遷移・採点・バッジ判定ロジック
- `data/categories.js` — 全13カテゴリのメタデータ
- `data/cat01_basic.js` — 基礎知識編100問・小枠(subset)定義
- `data/cat01_references.js` — 基礎知識編の参照(図解)テーブル

## 採点ロジック

- 内部では `correctCount * 100 / totalCount` の正確な小数値を保持し、表示のみ整数に繰り上げる
- 満点判定は内部の正確な値(correctCount === totalCount)で行い、表示用の丸め値には依存しない

## バッジ制度

小枠ごとの満点到達回数の累積で判定(連続満点は条件としない)。

| バッジ | 条件(満点到達回数) |
|--------|----------------------|
| GOOD | 1回 |
| GREAT | 2回 |
| FANTASTIC | 3回 |
