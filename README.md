# ドラマー知識検定(仮称)

ドラマー向け知識検定アプリ。暗記ではなく理解・判断・応用・表現を評価することを目指す。

## 現状(WEB版・全カテゴリ実装済み)

- 全13カテゴリ × 各100問 = **1,300問** を実装
- 各カテゴリは 初級30 / 中級25 / 上級25 / プロ20 の難易度構成、計24の小枠(subset)に分割
- 選択肢は「明らかな捨て回答」を排し、①正解 ②逆の意味 ③別の実在する技術・機材 ④架空の技術・機材 ⑤そもそも論 ⑥理論のすり替え ⑦機材加工での対応 ⑧誤った手順 ⑨専門用語の羅列 のパレットで構成
- 楽器・ブランド・ジャンルのルーツなど歴史的背景を各カテゴリに組み込み
- 画像を伴う問題(写真・イラスト・譜面)はアセット未調達のためプレースホルダー表示
- 広告(バナー/インタースティシャル)は実際の広告SDKと未連携のプレースホルダー実装(WEBローンチ後に収益化)

### カテゴリ一覧

1. 基礎知識編 / 2. 楽器本体編 / 3. 楽器パーツ編 / 4. ハードウェア編 / 5. セッティング編 / 6. メンテナンス編 / 7. 音作り編 / 8. 演奏技術編 / 9. 演奏表現・アレンジ編 / 10. コンビネーション編 / 11. トラブルシューティング編 / 12. 現場判断編 / 13. ドラマー思考編

## 動かし方

`index.html` をブラウザで直接開く(ビルド不要の静的サイト)。GitHub Pages で公開可能。

## ファイル構成

- `index.html` — エントリーポイント(全カテゴリのデータを読み込む)
- `styles.css` — スタイル
- `app.js` — 画面遷移・採点・バッジ判定・選択肢シャッフルロジック(全カテゴリ対応エンジン)
- `data/categories.js` — 全13カテゴリのメタデータ
- `data/catNN_*.js` — 各カテゴリの100問・小枠(subset)定義(NN = 01〜13)
- `data/catNN_references.js` — 各カテゴリの参照(図解)テーブル
- `data/_GENERATION_SPEC.md` — 問題生成仕様(選択肢パレット・難易度構成・検証手順)

## 採点ロジック

- 内部では `correctCount * 100 / totalCount` の正確な小数値を保持し、表示のみ整数に繰り上げる
- 満点判定は内部の正確な値(correctCount === totalCount)で行い、表示用の丸め値には依存しない

## バッジ制度

小枠ごとの満点到達回数の累積で判定(連続満点は条件としない)。

| バッジ | 条件(満点到達回数) |
|--------|----------------------|
| GOOD | 1回 |
| GREAT | 2回 |
| FANTASTIC | 3回 |
